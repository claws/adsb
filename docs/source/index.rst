Quick Start
===========

adsb is available on PyPI and can be installed with `pip <https://pip.pypa.io>`_.

.. code-block:: console

    $ pip install adsb

After installing adsb you can use it like any other Python module.

Here is a simple example:

.. code-block:: python

    import adsb
    # Insert common use-case.
